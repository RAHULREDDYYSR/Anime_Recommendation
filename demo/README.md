# Demo Screenshots

This folder contains screenshots of the Anime Recommendation System in action.

## Screenshots

1. **Home Page** (`01_home_page.png`) - Initial landing page with search input
2. **Search Query** (`02_search_query.png`) - User entering a query
3. **Loading State** (`03_loading.png`) - Parallel image loading in progress
4. **Recommendations** (`04_recommendations.png`) - 10 anime recommendations displayed
5. **Synopsis Expanded** (`05_synopsis.png`) - Full synopsis view
6. **Google Search Link** (`06_google_link.png`) - Clickable anime title

## How to Capture Screenshots

1. Run the Streamlit app: `uv run streamlit run app.py`
2. Navigate to `http://localhost:8501`
3. Take screenshots of:
   - Initial page
   - Entering query: "I want a shonen anime with good fights"
   - Click "Get Recommendations"
   - View the 10 recommendations with images
   - Expand a synopsis
   - Hover over an anime title to show the link

4. Save screenshots to this folder with descriptive names
5. Update the README.md to reference these screenshots
